# BLOCKCHAIN ASSIGNMENT 2 - AI MODEL MARKETPLACE

## Project Description
The AI Model Marketplace is a decentralized application (dApp) that allows users to list, purchase, and rate AI models. Built on blockchain technology, the application ensures transparency and fairness for our customers. The core functionalities are powered by a smart contract, while the frontend provides a user-friendly interface for interacting with the platform.

## Functionality
Functionality of the project mainly based on the implementation of five following methods:
- listModel() - To add a new AI model to the marketplace with a name, description, and price.
- purchaseModel() - To buy a specific AI model by its ID
- rateModel() - To rate purchased AI model
- withdrawFunds() - To withdraw accumulated funds from model sales.
- getMoreDetails() - To retrieve details of a specific AI model, including its name, description, price, creator, and average rating.
